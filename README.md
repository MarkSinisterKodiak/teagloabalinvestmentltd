## About

Swift Transport provides fast, reliable trucking and logistics solutions including:
- Full Truckload (FTL)
- Less Than Truckload (LTL)
- Express Delivery
- Secure Transport

## Tech Stack

- **React** - UI Framework
- **TypeScript** - Type Safety
- **Vite** - Build Tool
- **Tailwind CSS** - Styling
- **shadcn/ui** - UI Components

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
